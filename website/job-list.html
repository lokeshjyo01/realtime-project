AWS DevOps Engineer

Remote Full Time $90,000 - $120,000

Job Description

As an AWS DevOps Engineer, you will play a key role in implementing and managing AWS-based infrastructure, driving automation, and enhancing the deployment pipeline for our applications. You will work closely with software development and operations teams to ensure efficient delivery, scalability, and security of applications in a cloud environment.

Responsibilities

  • Design, build, and manage AWS infrastructure and DevOps pipelines.
  • Automate provisioning, configuration, and deployments using tools like Ansible and Terraform.
  • Collaborate with development teams to optimize CI/CD processes in Jenkins.
  • Ensure cloud security best practices and manage IAM roles and policies.
  • Monitor application performance and troubleshoot issues in production.
